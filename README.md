# Power-BI-Sales-Analytics-Dashboard-End-to-End-Data-Analysis-Project
Working with large datasets and transforming them into business insights is one of the most exciting parts of data analytics. This project strengthened my ability to create interactive dashboards that support data-driven decision making.

## 🚀 Project Overview
This project focuses on analyzing sales performance using a dataset containing *50,000 records. The goal was to transform raw data into meaningful business insights through an interactive **Power BI dashboard*.

The dashboard provides insights into *sales performance, profit trends, product categories, city-wise distribution, and customer ratings*.

---

## 📁 Dataset Information
Dataset Size: *50,000 rows*

Main Columns:
- Order ID
- Category
- City
- Sales
- Profit
- Rating
- Month

The dataset represents transactions across multiple product categories and cities.
